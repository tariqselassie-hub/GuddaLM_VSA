# Security Policy

Project: GuddaLM_VSA
Scope: GuddaLM_VSA only.

## Reporting a Vulnerability
Report vulnerabilities directly to the repository maintainer.
Do not file public issues for active security problems.

Preferred contact: Zoddjr@gmail.com
C/O Mr Terrence A. Jones Sr.

Include:
- Description of the issue.
- Steps or reproduction case.
- Potential impact.

## Response
- Acknowledge within a reasonable time.
- Provide status updates as the issue is assessed.

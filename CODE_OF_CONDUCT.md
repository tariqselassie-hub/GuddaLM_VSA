# Code of Conduct

This project follows the Contributor Covenant Code of Conduct, version 2.1.

Contact for code of conduct issues: guddalm@project.local

## Standards
- Be respectful and inclusive.
- Welcome newcomers and offer constructive feedback.
- Focus on what is best for the project and community.

## Unacceptable behavior
- Harassment, trolling, or personal attacks.
- Public or private harassment of any kind.

## Scope
Applies to project repositories, issue trackers, PRs, and official communications.
